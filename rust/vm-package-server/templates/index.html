Quick Setup

Automatic Setup (All Package Managers)

curl -fsSL http://localhost:3080/setup.sh | bash

PyPI

pip config set global.index-url http://localhost:3080/pypi/simple/

npm

npm config set registry http://localhost:3080/npm/

Cargo

echo '[registries.local] index = "http://localhost:3080/cargo/index"' >> ~/.cargo/config.toml

API Endpoints

POST /pypi/upload

npm

Cargo

PUT /cargo/api/v1/crates/new

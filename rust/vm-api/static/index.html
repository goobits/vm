Workspace Management

GET /api/v1/workspaces

List all workspaces for authenticated user

POST /api/v1/workspaces

Create a new workspace

GET /api/v1/workspaces/{id}

Get workspace details

DELETE /api/v1/workspaces/{id}

Delete a workspace

POST /api/v1/workspaces/{id}/start

Start a workspace

POST /api/v1/workspaces/{id}/stop

Stop a workspace

POST /api/v1/workspaces/{id}/restart

Restart a workspace

Snapshot Management

GET /api/v1/workspaces/{id}/snapshots

List all snapshots for a workspace

POST /api/v1/workspaces/{id}/snapshots

Create a new snapshot

POST /api/v1/workspaces/{id}/snapshots/{snapshot_id}/restore

Restore workspace from snapshot

Operations

GET /api/v1/operations

List operations with optional filters (workspace_id, type, status)

GET /api/v1/operations/{id}

Get operation details

Authentication Required: All API endpoints except /health require authentication via Bearer token in the Authorization header.
